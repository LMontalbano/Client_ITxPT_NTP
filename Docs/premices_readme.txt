### Explication du Projet :







### Explication de l'installation du logiciel :

- Récupération du projet via un git clone :

	- Déplacez vous dans le repertoire 'Tests_Services_ITxPT/App/dist'
	- Voici le logiciel !!! (main.exe)
	- Vous pouvez vous embêter à faire tout le chemin à chaque fois pour accéder au main.exe
	- Ou alors créer un raccourci sur le bureau, clic droit 'Envoyer vers' -> 'Bureau (créer un raccourci)'
	- Vous pouvez même changer le nom du raccourci à votre bon vouloir ! clic droit 'Renommer' (par exemple : Tests_Services_ITxPT)


- Récupération du projet via le téléchargement d'un ZIP :
	
	- Direction 'Téléchargement'
	- Unziper le dossier Tests_Services_ITxPT.zip
	- Déplacez vous dans le repertoire 'Tests_Services_ITxPT/App/dist'
	- Voici le logiciel !!! (main.exe)
	- Vous pouvez vous embêter à faire tout le chemin à chaque fois pour accéder au main.exe
	- Ou alors créer un raccourci sur le bureau, clic droit 'Envoyer vers' -> 'Bureau (créer un raccourci)'
	- Vous pouvez même changer le nom du raccourci à votre bon vouloir ! clic droit 'Renommer' (par exemple : Tests_Services_ITxPT)




### Explication de l'utilisation du logiciel :

Il y à différents élements sur la fenêtre principale du logiciel, nous allons les détailler si dessous:

	- Deux inputs :
		- Local Address : l'adresse ip local de la machine
		- SAE Address : l'adresse ip de l'UC SAE
	
	- Le logo de Keolis Orléans Métropole

	- Six boutons:
		- Un bouton 'Test NTP' : Il permet de lancer 5 itérations du Test NTP
		- Un bouton 'Test GNSS' : Il permet de lancer 5 itérations du Test GNSS
		- Un bouton 'Test AVMS' : Il permet de lancer le Test AVMS
		- Un bouton 'Cancel Test AVMS' : Il permet de cancel le Test AVMS
		- Un bouton 'Test APC' : Il permet de lancer le Test APC
		- Un bouton 'All Test' : Il permet de lancer 5 itérations de chaque test les un après les autres

	- Une fenêtre d'affichage
	
	- Le lien github du projet
